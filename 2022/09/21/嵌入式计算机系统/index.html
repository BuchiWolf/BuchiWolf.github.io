第二讲 硬件基础

封装：

  • 双列直插式封装（DualIn-line package DIP）

    中小规模集成电路均采用这种封装形式

    适合在PCB（印刷电路板）上穿孔焊接，操作方便

    芯片面积与封装面积之间比值较大，故体积也较大

    引脚数≤100

  • 扁平式封装（PQFP/PFP）

    大规模/超大型集成电路采用这种封装形式

    采用SMT（表面安装设备技术）表贴到PCB

  • 薄型小外形封装（Thin Small Outline Package TSOP）

    常用于内存存储器

    在芯片的周围做出引脚，采用表面安装技术SMT直接附着在PCB表面

    寄生参数（电流大幅度变化时，引起输出电压扰动）减小，适合高频应用，操作方便

  • 塑料引脚芯片载体封装（PLCC）

    匹配专门的安装卡座，可机械插拔

    主板（如Intel系列）的BIOS，或BIOS固件封装的常用形式

    外形尺寸比DIP小

  • 插针网格阵列封装（Pin Grid Array Package PGA）

    PGA插座，专门用来满足封装安装要求

  • 球栅阵列封装（Ball Grid Array Package BGA）

    多用于CPU，主板上南北桥芯片等高密度高性能多引脚封装

    信号传输延迟小，适应频率高

数据表 印制电路板（PCB）

电源地与信号地

电源地主要是针对电源回路电流所走的路径而言的，一般来说电源地流过的电流较大，而信号地主要是针对两块芯片或者模块之间的通信信号的回流所流过的路径，一般来说信号地流过的电流很小。

其实两者都是GND，之所以分开来说，是想让大家明白在布PCB板时要清楚地了解电源及信号回流各自所流过的路径，然后在布板时考虑如何避免电源及信号共用回流路径，如果共用的话，有可能会导致电源地上大的电流会在信号地上产生一个电压差（可以解释为：导线是有阻抗的，只是很小的阻值，但如果所流过的电流较大时，也会在此导线上产生电位差，这也叫共阻抗干扰），使信号地的真实电位高于0V，如果信号地的电位较大时，有可能会使信号本来是高电平的，但却误判为低电平。

当然电源地本来就很不干净，这样做也避免由于干扰使信号误判。所以将两者地在布线时稍微注意一下，就可以。一般来说即使在一起也不会产生大的问题，因为数字电路的门限较高。除了正确进行接地设计、安装,还要正确进行各种不同信号的接地处理。

电路供电电压Vcc > 芯片工作电压Vdd，发射级电源电压Vee，编程电压Vpp

信号：信息的物理表示

信号浮动（多驱动信号）：安装辅助元件来辅助其高低：上拉电阻、下拉电阻

三态门：可以将信号驱动为高电压、低电压、浮动状态（高阻抗态）

连接在元器件信号引脚和高电压之间的电阻，用于将引脚信号钳位在高电平，或在驱动能力不足时提供电流。

通常其值可在1KΩ～100KΩ之间，值越大输出电平的延迟越大；值越小，电流越大，上拉能力越强，但损耗越大

下拉电阻连接在元器件信号引脚和GND之间的电阻，用于将信号钳位在低电平，或者用于吸收电流。

上拉就是将不确定的信号通过一个电阻钳位在高电平，电阻同时起限流作用。下拉同理，也是将不确定的信号通过一个电阻钳位在低电平。

上拉是对器件输入电流，下拉是输出电流；强弱只是上拉电阻的阻值不同，没有什么严格区分；对于非集电极（或漏极）开路输出型电路（如普通门电路）提供电流和电压的能力是有限的，上拉电阻的功能主要是为集电极开路输出型电路输出电流通道。

负载问题

  • 一个元件的输出驱动能力是有限的，当一个输出对应过多元件的输入，可能导致驱动过载
  • 制造商已标明元件输出信号上的驱动电流及所需的输入电流
  • 当出现驱动过载现象时需要增加驱动器电路

解决方法

  • 增加驱动器：但是会引入延时
  • 分到不同的线路上：增加器件也会引入延时

基本元件与逻辑

  • 晶体管：Transistor，固体半导体制成的器件
    • 具有检波、整流、放大、开关、稳压、信号调制等多种功能；二极管（Diode）晶体管有两个极，一个PN结，单向导电特性

  • 三极管：双极性结型二极管

  1. 退耦电容

  • 电路中的耦合是指电路网络中电流大小发生变化时，在其他电路网络中感应产生的寄生电流或供电电路中形成的反冲电流。

  • 当芯片改变输出端电压时，需要大量的电量改变输出信号，此时芯片供电电路电压会急剧下降，可能导致芯片停止工作。

  • 退耦（也成为去耦）的目的就是消除电路之间的寄生耦合（退噪声，储能）。

    • 阻断不期望的信号
    • 基于储能快速提供能量补充

  • 在电路特定位置接入电容可达到电路退耦的效果，该类电容就被称为退耦电容

  • 布放在信号输出引脚，将输出信号的干扰作为滤除对象

  • 一般分布在电路板上离可能需要大量点亮的电路部分很近的地方

    • 最好不采用电解电容，这种结构在高频时表现为电感

  • 布放规则：靠近相应引脚（电源或信号），电容的电源或信号走线与地线所包围的面积最小

  1. 旁路电容
  • 在电路中也是用于滤波，抗干扰的作用
  • 工作原理是将输入信号中的高频噪声作为滤除对象，利用电容的频率阻抗特性，将前级携带的高频杂波滤除
  1. 寄存器
  • 寄存器是一种基于触发器逻辑构造的高速存储单元
  • 用于暂存计算过程中的指令、数据和地址
  1. 计数器
  • 集成电路中完成计数功能的逻辑部件
  • 由一组具有存储信息功能的触发器构成
  • J-K触发器 R-S触发器 T触发器 D触发器
  • 当输入为系统时钟时，则成为定时器
  1. 总线
  • 连接各种功能部件并传输信息的一组通信线路
  • 根据传递的信息，可将其分为数据总线，地址总线和控制总线

第三章 嵌入式处理器

3.1.1 处理及其本组成模型

处理器是从存储器中逐条获取指令并进行解释、执行的硬件引擎

控制单元包括了PC/IR/ID，采用高速缓存来提高访问速度，同时还需要采用堆栈来解决中断、函数调用返回时的现场恢复问题。

执行单元则主要由ALU/DR/FR组成。

在处理器体系中，一个控制单元和一个执行单元即可组成完整、独立的指令执行引擎，通常称为计算核（Core）。处理器中计算核的数量则是指其EU的数量，每个EU至少需要一个CU的支持。

座：物理cpu个数

通常情况下。一条指令由表示运算方式的机器操作码（Opcode）以及零个或多个算子组成，而可执行程序则是顺序存储的一组机器码的集合
